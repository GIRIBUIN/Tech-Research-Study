# Tech Research Study

## 규칙

❌ 논문 원문 PDF나 유료 자료 업로드 금지

✅ 논문, 공식 문서, 기사, 블로그 등 모든 참고 자료는 URL로 정리하고 출처를 남기기

❌ AI에게 최종 Markdown/PDF 문서를 대신 생성하게 하는 행위 금지

❌ 원문 내용을 그대로 복붙하는 행위 금지

✅ AI를 공부하고 이해하는 보조 도구로 사용하는 것은 적극 장려

---

## 1. 스터디 목적

단순히 자료를 모으는 것이 아니라, 기술을 조사하고 이해한 내용을 자신의 언어로 정리한 뒤 스터디원과 공유하는 것.

주요 목표

- 최신 기술의 개념과 구조 이해
- 기술이 등장한 배경과 문제의식 파악
- 실제 활용 사례와 한계 분석
- 관련 논문, 공식 문서, 기술 블로그 읽기
- 조사 내용을 바탕으로 토론 질문 만들기
- 개인적인 해석과 생각의 변화 기록

---

## 2. 진행 방식

- 매주 토요일까지 각자 주제 업로드
- 매주 일요일까지 다른 사람이 공유한 내용에 코멘트 남기기
- 모든 스터디원이 최소 1개 이상의 코멘트를 남긴 후 PR을 merge

진행 순서

1. 매주 하나의 주제 선정하기
2. 해당 주제에 대해 공부하기
   - 최신 기술
   - 논문
   - 공식 문서
   - 기사
   - 기술 블로그
3. 공부 내용을 Markdown 또는 PDF로 직접 정리해서 공유하기
4. 다른 사람의 정리 내용에 코멘트 남기기
5. 코멘트 내용을 바탕으로 필요하면 수정하기
